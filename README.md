# AI-powered Call Auditing Tool

An AI-powered call auditing tool designed to empower agents by providing detailed insights and scores for their calls. This application leverages cutting-edge AI technologies to automate the call review process.

## Built With

*   **Firebase**: For backend services, authentication, and database.
*   **Google Gemini**: For advanced AI-driven call analysis and scoring.
*   **AssemblyAI**: For accurate transcription of call recordings.
*   **Next.js**: Frontend framework for building the user interface.
*   **React**: JavaScript library for building user interfaces.

## Features

*   **Secure User Authentication**: Protects access to the auditing tool.
*   **Call Recording Uploads**: Easily upload audio files for auditing.
*   **Metadata Management**: Associate calls with relevant information such as university, domain, call type, and date.
*   **AI-Powered Processing**: Automatic transcription and analysis of calls.
*   **Detailed Call Scoring**: Provides scores across various criteria (c1-c10 as seen in common.ts).
*   **Intuitive User Interface**: Designed for agents and administrators to manage and review audits.
