# **App Name**: Call Quality Compass

## Core Features:

- Audio Preparation via (Assembly AI API) ASR: Transcribe audio, separate voices, measure pauses, holds, and audio quality.
- LLM Audio Check: (Gemini 2.5 Pro API): AI tool to analyze transcripts and audio metrics to assess agent behavior, providing objective feedback.
- Call Upload & Info: Upload call audio files, enter university name, domain, and call date.
- Rubric Scoring: Apply a standardized rubric to the call with defined criteria.
- Coaching Tip Generator: Generate coaching tips based on rubric scores, referencing time stamps of notable parts of the audio.
- Record Storage: Store key call data, including transcript, scores, notes, and audit date.
- Export to Google Sheet: Export compiled results into a structured Google Sheet.

## Style Guidelines:

- Primary color: Soft blue (#7BBEE3) to evoke trust, clarity, and a calm assessment environment.
- Background color: Light gray (#F0F4F8) for a neutral and unobtrusive backdrop that highlights content.
- Accent color: Warm orange (#E68A2E) to draw attention to important actions and AI-driven insights.
- Body and headline font: 'PT Sans', a humanist sans-serif for modern and accessible readability.
- Use minimalist, professional icons representing data analysis and call metrics.
- Clean and structured layout with clear section divisions for each stage of the CQC process.
- Subtle transitions and progress indicators during audio processing and AI analysis stages.
